# Instalación Beego y Bee

- Necesita una versión de [Go 1.1 + instalada](golang.md)
- Necesita tener git instalado

  ```bash
  go get -u github.com/astaxie/beego
  go get -u github.com/beego/bee
  ```
- Comprobar instalacion

  ```bash
  $ bee version
  ```
